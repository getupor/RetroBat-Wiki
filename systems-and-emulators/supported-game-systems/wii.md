## Controls

The mapping hereunder is explained for the Classic Controller.

| RetroBat key                                                                        | Classic Controller key |
| ----------------------------------------------------------------------------------- | ---------------------- |
| START                                                                               | +                      |
| SELECT / BACK                                                                       | -                      |
| D-PAD                                                                               | D-PAD                  |
| Left Analog Stick                                                                   | Left Stick             |
| Right Analog Stick                                                                  | Right Stick            |
